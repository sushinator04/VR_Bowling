VR Bowling 

A complete VR bowling game built in Unity using the XR Interaction Toolkit and SteamVR.*

VR Bowling is a fully playable **Virtual Reality bowling game** created in Unity.
It features realistic bowling physics, accurate throw mechanics, interactive pin behavior, automated scoring, and complete room-scale VR interaction.

---

##  Features

### **Bowling Mechanics**

* Realistic ball physics using velocity-based throwing
* Fully functioning pin collisions and knockdown detection
* Complete scoring system (frames, strikes, spares)
* Automatic lane & pin reset system

###  **VR Interaction**

* Grab & throw bowling balls using XR Interaction Toolkit
* Accurate controller-tracked input
* Smooth room-scale locomotion (SteamVR)
* Natural hand interactions with VR controllers

###  **Technical Highlights**

* Built with **Unity 2022.3.24f1 (LTS)**
* Uses **XR Interaction Toolkit** for modern VR input
* SteamVR plugin for runtime deployment
* Optimized colliders & rigidbody physics
* Modular prefabs (balls, pins, lane, resetter)
* Clean scene structure for extension or multiplayer expansion

---
